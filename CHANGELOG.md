## [0.2.0] - 2020-09-08
- git integration: SimpleCov groups are automatically generated based on branch
  and commit metadata.
  ([#2](https://github.com/ticky/simplecov-buildkite/pull/2),
  [#8](https://github.com/ticky/simplecov-buildkite/pull/8))
